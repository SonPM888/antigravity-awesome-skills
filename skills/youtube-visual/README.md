# YouTube Visual Architect V1.0 (The Cinematic Engine) 🎨

Hệ thống kiến trúc hình ảnh chuyên nghiệp dành cho YouTube, chuyên biến kịch bản chữ (VoiceScript) thành các Image Prompt (Prompts tạo ảnh) mang tính ẩn dụ, nghệ thuật và sẵn sàng để tự động hóa sản xuất.

## 🌟 Tại sao nên dùng Skill này?

`youtube-visual` được thiết kế để giải quyết bài toán khó nhất của video Faceless: **"Làm sao để ảnh đẹp lung linh, đồng nhất nhân vật và thực sự chạm tới cảm xúc?"**

1.  **Tư duy Ẩn dụ (Metaphor):** Không chỉ minh họa bề mặt, Skill tìm kiếm các hình ảnh biểu tượng để tạo ra sự tò mò và cảm xúc sâu sắc cho khán giả.
2.  **Đồng nhất thị giác (Visual DNA):** Tự động "tiêm" các thông số nhân vật và phong cách vào từng Prompt, giữ cho 100+ ảnh luôn đồng nhất về kiểu dáng.
3.  **Tối ưu cho Tự động hóa (Automation-Ready):** Đầu ra được định dạng sẵn dưới dạng Python List hoặc JSON, giúp bạn dán trực tiếp vào các script Python sử dụng **Runware API** mà không cần sửa đổi thêm.
4.  **Nhịp điệu điện ảnh:** Áp dụng quy tắc đảo góc máy chuyên nghiệp, giúp video luôn động và hấp dẫn.

## 🛠️ Quy trình thực hiện 6 Bước

- **Phase 0 (Branding):** Trích xuất thông số từ ảnh nhân vật và ảnh phong cách mẫu.
- **Phase 1 (Semantic Mapping):** Chia kịch bản thành các cảnh dựa trên ý nghĩa (1-2 câu/cảnh).
- **Phase 2 (Metaphor Lab):** Tạo ra các ý tưởng ẩn dụ đắt giá cho từng phân đoạn.
- **Phase 3 (Rhythm Mapping):** Gán các góc máy (Close-up, Wide, POV...) để đảm bảo nhịp điệu thị giác.
- **Phase 4 (Prompt Alchemy):** Chế tác Prompt hoàn chỉnh, sạch sẽ cho AI (Flux, SDXL, Runware).
- **Phase 5 (Master Export):** Xuất file dữ liệu sẵn sàng để dán vào Code thực thi.
