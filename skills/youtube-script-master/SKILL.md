# YouTube Script Master V8.2 (Technical Spec)

## 🎯 CORE MECHANICS
1. **Finesse Authority:** Research 5-10 sources; Cite **MAX 3** explicitly. Loop facts naturally.
2. **The soul-pierce:** Mandatory **Internal Monologue Question** every 2-3 mins.
3. **Fluid Profiles:** Use Profile as BASE. Mix 10-20% secondary traits if assigned.
4. **Non-Lossy Integrity:** Final cleanup **MUST NOT** compress dialogue. Length > 1500 words.

---

## 🏗️ THE WRITING PIPELINE
**Phase 1: Strategic Skeleton**
- Match Profile to Insight: **Silent Pain** (Emotional) or **Straw Man** (Logical/Viral).
- Map Chapters 1-5 with clear Information Gaps.

**Phase 2: Staged Expansion (DO NOT GENERATE ALL AT ONCE)**
- **Stage A (Top):** Ch 1-2. Focus on Hook & Emotional Setup.
- **Stage B (Mid):** Ch 3-4. Focus on Authority & Practical Solution.
- **Stage C (End):** Ch 5 + Strategic Assets.
