# YouTube Script Quality Auditor V1.1 (Streamlined Edition)

This skill performs an independent, comprehensive quality audit on any YouTube script to ensure it meets the V5.4 Clarity Edition standards.

---

## 🎯 PURPOSE

Use this skill AFTER a script has been generated to:
1. Validate adherence to V5.4 quality standards
2. Identify gaps in sensory immersion, cultural authenticity, and asset completeness
3. Provide actionable recommendations for improvement

---

## 📋 THE 5-GROUP AUDIT FRAMEWORK

### **Group 1: Structure & Foundation** ✅
**Checks:**
- Word count: 1500-1800 words (±100 acceptable)
- Metaphor consistency: Appears in 3+ chapters
- CTA altruism: Framed as helping others

**Pass Criteria:** All 3 checks pass

---

### **Group 2: Sensory Immersion** 🌈
**Checks:**
- **Density:** 1 description per 100 words (e.g., 1500 words = 15 descriptions)
- **Diversity:** At least 2 different senses per chapter
- **Balance:** <70% visual/auditory (prioritize smell, taste, touch)

**Examples:**
- ✅ Good: "Mùi cà phê (smell), vị đắng (taste), hơi ấm chạm má (touch)"
- ❌ Bad: "Nhìn thấy cà phê, nghe tiếng pha chế" (only visual + auditory)

**Pass Criteria:** All 3 checks pass

---

### **Group 3: Story Authenticity** 📖
**Checks:**
- **Setting:** Everyday contexts (home, park, community center, public hospital)
- **Solution Source:** Self-driven OR common sources (radio, friend, family)
- **Cultural Cliché:** No "savior figures" (monks, famous doctors) unless culturally appropriate

**Red Flags:**
- Remote/luxury settings (mountain temples, resorts)
- Miraculous coincidences
- Celebrity experts appearing out of nowhere

**Examples:**
- ✅ Good: Character learns from morning radio while making coffee
- ❌ Bad: Character meets renowned monk at remote temple

**Pass Criteria:** All 3 checks pass

---

### **Group 4: Asset Completeness** 📦
**Required Assets (V5.2+):**
1. ✅ 2 Hook Variations (Emotional + High-Stakes)
2. ✅ Chapter Markers (Time-indexed)
3. ✅ Typography List (5-7 quotes)
4. ✅ Viral Shorts Script (60s with visual loop)
5. ✅ Community Post (Text + Poll)

**Shorts Visual Loop Check:**
Must include: Hand/finger + Direction + Speed + Repetitions
- ✅ Good: "Ngón cái tay phải, vòng tròn theo chiều kim đồng hồ, 3 vòng/giây, 50 lần"
- ❌ Bad: "Bấm huyệt nhiều lần"

**Pass Criteria:** All 5 assets present + Shorts has visual loop

---

### **Group 5: Profile Compliance** 🛡️
**For Seniors Profile:**
- No deficiency language ("fix your problem," "you're broken")
- Agency restoration ("you may consider," "it's within your power")
- Honorary terms correct (Bác/Cô/Chú for Vietnamese)

**For Office Warriors Profile:**
- Professional tone maintained
- No condescension
- Strategic frameworks present

**Pass Criteria:** No violations found

## 📊 SIMPLIFIED AUDIT REPORT FORMAT

After running the audit, provide a report in this format:

```
# QUALITY AUDIT REPORT - [Script Title]
Auditor: YouTube Script Quality Auditor V1.1 | Date: [Date]

## 🎯 OVERALL SCORE: X/5 Groups ⭐

### ✅ PASSED GROUPS (X/5)
- Group 1: Structure & Foundation ✅
- Group 4: Asset Completeness ✅

### ⚠️ FAILED GROUPS (X/5)
**Group 2: Sensory Immersion** ❌
- Issue: Only 11 descriptions (target: 15)
- Location: Chapters 1, 3, 4 lack smell/taste
- Fix: Add 4 olfactory/gustatory details in identified chapters

**Group 3: Story Authenticity** ⚠️
- Issue: Character meets monk at temple (cultural cliché)
- Location: Chapter 3, lines 104-110
- Fix: Replace with radio health show scenario

### 🎯 PRIORITY FIXES (Top 3)
1. [Most critical] Add 4 sensory descriptions (smell/taste)
2. [Important] Replace monk story with everyday scenario
3. [Minor] Add pressure detail to Shorts Script

### ✅ CERTIFICATION
[✅ CERTIFIED V5.4] or [⚠️ REQUIRES REVISION - X issues]
```

---

## 🔧 USAGE INSTRUCTIONS

1. **Input:** Paste the complete script text
2. **Execution:** Auditor checks all 5 groups sequentially
3. **Output:** Receive simplified report with specific fixes
4. **Iteration:** Make fixes and re-audit if needed

---

## 🚨 CRITICAL RULES

- **Independence:** Operates independently of script generation
- **No Auto-Fix:** Only reports issues, does NOT fix them
- **User Decision:** User decides which recommendations to implement
- **Re-Audit:** Script can be re-audited after fixes

---

## 💡 WHEN TO USE THIS SKILL

- After generating any YouTube script with `youtube-script-master`
- Before sending a script to production (voice recording, video editing)
- When you want to validate an older script against V5.4 standards
- When you need an objective quality assessment
